# CoolDrivePro Search Console Checklist

Use this checklist after each meaningful SEO release and after each new batch of indexable pages.

## 1. Property And Verification

- [ ] Confirm the correct production property is added for `https://cooldrivepro.com/`.
- [ ] Verify ownership at the domain or URL-prefix level.
- [ ] Confirm the sitemap URL resolves publicly at `https://cooldrivepro.com/sitemap.xml`.

## 2. Initial Submission

- [ ] Submit `https://cooldrivepro.com/sitemap.xml` in Search Console.
- [ ] Confirm the submitted sitemap is accepted without fetch errors.
- [ ] Check that the sitemap shows the new commercial routes:
  - `/solutions/semi-truck-parking-ac/`
  - `/solutions/rv-parking-ac/`
  - `/solutions/van-parking-ac/`
  - `/compare/12v-vs-24v-parking-ac/`
  - `/compare/rooftop-vs-mini-split-parking-ac/`
  - `/compare/parking-ac-battery-runtime/`
  - `/compare/cooling-only-vs-heating-cooling-parking-ac/`
  - `/compare/parking-ac-roof-fitment-guide/`

- [ ] Page is allowed by `robots.txt`.
- [ ] Indexing is allowed.
- [ ] Referring page and sitemap entry are present.

## 4. Indexing Diagnostics

Check the Page Indexing report after submission.

- [ ] No unexpected `Alternate page with proper canonical tag` issues on commercial pages.
- [ ] No `Discovered - currently not indexed` buildup on the new hub pages.
- [ ] No `Crawled - currently not indexed` buildup on product and hub pages.
- [ ] No `Duplicate without user-selected canonical` issues across product, blog, and compare pages.
- [ ] No accidental `Excluded by noindex` on indexable commercial pages.

## 5. Enhancements And Rich Results

- [ ] Confirm Product rich result eligibility for the main product pages.
- [ ] Confirm FAQ-rich-result eligibility where FAQ schema exists.
- [ ] Confirm no structured-data parsing errors for the new compare pages.

## 6. Performance Monitoring

Review the Performance report after the next crawl/index cycle.

- [ ] Filter by page and check impressions for the new `solutions` pages.
- [ ] Filter by page and check impressions for the new `compare` pages.
- [ ] Filter by page and check impressions for the priority commercial-intent blog URLs listed above.

## 7. Ongoing Cadence

Weekly for the next 4 to 6 weeks:

- [ ] Recheck sitemap processing status.
- [ ] Recheck indexing status for the new commercial pages.
- [ ] Compare impressions and average position for the new pages.
- [ ] Identify pages getting impressions but weak CTR, then refine title and meta description.
- [ ] Identify pages with no impressions, then strengthen internal links or expand topical coverage.

## 8. Release Acceptance Standard

Treat the SEO release as operationally accepted when all of the following are true:

- [ ] Sitemap accepted and reachable.
- [ ] New product and commercial hub URLs inspected successfully.
- [ ] No unexpected indexing exclusions on target pages.
- [ ] Product catalog and product detail pages are internally linking to relevant hub pages.
- [ ] At least some of the new pages start receiving impressions in Search Console.
